## 📱 چگونه استفاده کنم؟

۱. روی دکمه بالا کلیک کن
۲. با GitHub لاگین کن
۳. پروژه ساخته می‌شود
۴. منتظر باش تا دیپلوی کامل شود
۵. به تب `Logs` برو و لینک پروکسی را کپی کن
۶. لینک را در تلگرام باز کن
